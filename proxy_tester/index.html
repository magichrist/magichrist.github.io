Proxy Tester

Proxy Tester is an open-source proxy testing tool designed to analyze VLESS, VMESS, Trojan, and Shadowsocks proxy links. It helps users evaluate ping, latency, speed, and connection stability before using a proxy in production or research environments.

Supported Proxy Protocols

  • VLESS
  • VMESS
  • Trojan
  • Shadowsocks

Key Features

  • Ping and latency measurement
  • Proxy speed and connectivity testing
  • Batch testing of proxy configuration files
  • CLI-friendly and automation-ready
  • Designed for security labs and network diagnostics

Who Is This For?

Proxy Tester is useful for security researchers, network engineers, penetration testers, VPN users, and developers who need to validate proxy configurations quickly and reliably.
